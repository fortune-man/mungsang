아르바이트 다녀왔다는 핑계로 아무것도 제대로 못함 ㅠ
## 1. 멍상이 만드는 밸류
- 몰입력 회복
- 창의력 발휘

## 2. 멍상 유저 플로우
1. 멍상 도메인 (www.spaceout.com)접속
2. 웰컴 페이지 - 멍상이 필요한 이유 안내하는 역할
	1. 일상 속 몰입력 회복 - "사장님 몰래 터득하는 잘 쉬는 방법"
	2. 창가 자리가 비싼 이유
	3. 멍 때리기 대회 사례
	4. 멋진 풍경을 감상하는 것의 두뇌 효과
	5. 멍상 서비스 핵심안내 - 쉼에 대한 정보를 모으는 서비스
3. 탭 스위칭 크리에이팅 / 고르기 (검색 / 조회)
4. 크리에이팅 - 내가 원하는 배경화면 제작
5. 검색 / 조회 - 마음에 드는 배경화면을 리스트 아이템 클릭하여 선택
6. 다운로드 하여 배경화면 설정 가능함을 보여주는 데모
7. 다운로드 완료 후 고객 만족 점수 받음
8. 다운로드된 유저 수 자동집계 하여 고객 창출 카운트
9. 나의 몰입력은 얼마나 회복되었을까?

## 3. 구현해야할 기능 목록
- [] 웰컴 페이지
	- [] 멍상이 필요한 이유에 대한 정보를 스크롤 할 수 있는 거대한 화면 배치
- [] 스위칭 할 수 있는 탭 - 크리에이팅 / 고르기 (검색 / 조회)
- [] 크리에이팅
	- [] 색상 선택
	- [] 드로잉 도구 선택
	- [] 다운로드
- [] 고르기 (검색 / 조회)
	- [] 검색
	- [] 검색 결과 조회
	- [] 선택한 리스트 아이템 확대
	- [] 다운로드 하여 배경화면 설정 가능함을 보여주는 데모
	- [] 다운로드 완료 후 고객 만족 점수 수집
	- [] 다운로드 수 자동 집계 -> 고객 창출 카운트
- [] 나의 몰입력은 얼마나 회복되었을까?
	- [] "준비중"
	- [] 사전신청
	- [] 사전신청 count
